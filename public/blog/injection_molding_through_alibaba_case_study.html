The Road to Gulpo

  1. Identify Suppliers
  2. Initial Reachout (do they exist)
  3. Create RFQ Package (with scrubbed file)
  4. Send RFQ Package
  5. Evaluate Quotes for Validity
  6. Order Samples
  7. Evaluate Samples
  8. Ask for buyer references (see if there is anyone else who has used them we can speak with)
  9. Enter into NDA with supplier (optional)
  10. Re-quote with final file
  11. If good, negotiate finance terms (e.g. establish escrow, % payment up front, etc)
  12. Order first article (mold fee, QTY 10 units, shipping of first batch)
  13. Evaluate first article
  14. Resolve outstanding balance with SupplyBetter
  15. Order rest of set with Ever Greater (50% deposit , 50% before delivery)
  16. Profit
